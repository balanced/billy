Billy - The open source recurring payment system
================================================

Billy is an open source recurring payment system. It can schedule 
charging/paying out transactions periodically for you. It is powered
by `Balanced Payments`_. Feel free to fork on GitHub_.

.. _GitHub: https://github.com/balanced/billy
.. _`Balanced Payments`: http://balancedpayments.com
